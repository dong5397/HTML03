html 태그의 종류는 다음과 같습니다

  • br 태그는 줄 바꿈할 때 사용한다.
  • blockquote 태그는 출처를 인용할 때 사용한다.
  • q 태그는 문단 안에 텍스트 단위의 짧은 안용문을 작성할 때 사용한다.
  • ins 택그는 새로 추가된 텍스트임을 나타낼 때 사용한다.
  • del 태그는 기존에 있던 텍스트가 삭제된 텍스트임을 나타낼 때 사용한다.
  • sub 태그는 아래첨자 H2
  • sup 태그는 위 첨자 입니다. 42는 16입니다.
  • div 태그는 블록 요소와 인라인 요소를 그룹으로 묶을 때 사용한다.
  • span 태그는 인라인 요소를 그룹으로 묶을 때 사용한다.
    • ul태그는 순서가 없는 비순서 목록을 생성할때 사용한다.
    • ol태그는 순서가 있는 순서 목록을 생성할때 사용한다.
  • dl태그는 정의형 목록을 생성할때 사용한다.
  • a 태그는 HTML내부나 외부 링크를 생성합니다
  • img 태그는 이미지 객체를 삽입하고 싶을 때 사용한다
  • strong 태그는 텍스트의 의미를 강조하고 싶을 때 사용한다
  • em 태그는 기울어진 효과를 낼 수 있다
  • form 태그는 폼 양식을 의미하는 태그입니다

p 태그로 작성된 영역.         중요한 부분은 굵게 기울리게 강조할 수 있다. 줄 바꿈은 마음대로 잘 되지 않습니다.
줄바꿈 태그를 사용해야지만 됩니다. 밑줄은 밑 태그 (u)를 사용할수 있습니다

하이퍼 텍스트 마크업 언어(영어: Hyper Text Markup Language, HTML, 문화어: 초본문표식달기언어, 하이퍼본문표식달기언어)는 웹 페이지 표시를 위해 개발된 지배적인 마크업 언어다. 또한, HTML은 제목, 단락, 목록 등과 같은 본문을 위한 구조적 의미를 나타내는 것뿐만 아니라 링크, 인용과 그 밖의 항목으로 구조적 문서를 만들 수 있는 방법을 제공한다. 그리고 이미지와 객체를 내장하여 대화형 양식을 생성하는 데 사용될 수 있다. HTML은 웹 페이지 콘텐츠 안의 꺾쇠 괄호에 둘러싸인 "태그"로 되어있는 HTML 요소 형태로 작성한다. HTML은 웹 브라우저와 같은 HTML 처리 장치의 행동에 영향을 주는 자바스크립트, 본문과 그 밖의 항목의 외관과 배치를 정의하는 CSS 같은 스크립트를 포함하거나 불러올 수 있다. HTML과 CSS 표준의 공동 책임자인 W3C는 명확하고 표상적인 마크업을 위하여 CSS의 사용을 권장한다.[1]
